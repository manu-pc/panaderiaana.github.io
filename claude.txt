Tengo una página web estática en español compuesta por varios archivos HTML en la raíz del proyecto.

Quiero convertirla en una web multilingüe (español, inglés y gallego) con buena estructura SEO internacional y selector de idioma visible con iconos de banderas.

Realiza lo siguiente:

1. REESTRUCTURACIÓN DE CARPETAS

- Mueve todos los HTML actuales (español) a una carpeta /es
- Crea carpetas adicionales:
  /en
  /gl

- Mantén la misma estructura de archivos en cada idioma.
- Si existen carpetas como /css, /js, /img, déjalas compartidas en la raíz.

Estructura final esperada:

/
  /es
    index.html
    pagina2.html
  /en
    index.html
    pagina2.html
  /gl
    index.html
    pagina2.html
  /css
  /js
  /img

2. TRADUCCIÓN

- Traduce todo el contenido visible al usuario al inglés y al gallego.
- NO traduzcas:
  - rutas
  - nombres de clases
  - ids
  - scripts
  - metadatos técnicos no visibles

- Mantén el significado natural y profesional (no traducción literal robótica).
- Mantén correctamente acentos y caracteres especiales.
- Conserva la indentación y formato limpio del HTML.

3. SEO INTERNACIONAL (MUY IMPORTANTE)

En cada archivo HTML:

- Añade atributo correcto en <html>:
  Español: <html lang="es">
  Inglés:  <html lang="en">
  Gallego: <html lang="gl">

- Añade dentro de <head> las etiquetas hreflang cruzadas:

Ejemplo para /es/index.html:

<link rel="alternate" hreflang="es" href="https://midominio.com/es/" />
<link rel="alternate" hreflang="en" href="https://midominio.com/en/" />
<link rel="alternate" hreflang="gl" href="https://midominio.com/gl/" />
<link rel="alternate" hreflang="x-default" href="https://midominio.com/es/" />

- Añade canonical correcto para cada idioma.
- Traduce también:
  - <title>
  - <meta name="description">
  - textos ALT de imágenes

4. SELECTOR DE IDIOMA CON BANDERAS

Añade en el header un selector visible con iconos de banderas, usando los .svg correctos de bandera que hay en /img/.


Requisitos:
- Que mantenga la misma página al cambiar idioma.
  Ejemplo:
    /es/pagina2.html
    /en/pagina2.html
    /gl/pagina2.html

- Usa rutas relativas correctas.
- No uses JavaScript para cambiar idioma (solo enlaces normales).
- Añade atributo hreflang también en los enlaces del selector.

5. CALIDAD Y LIMPIEZA

- Devuélveme:
  1) Nueva estructura de carpetas
  2) Un ejemplo completo de un archivo index.html en los tres idiomas
  3) Lista de cambios realizados
  4) Posibles mejoras SEO adicionales

- No rompas rutas existentes.
- No elimines scripts ni estilos.
- No simplifiques contenido.

Trabaja como un desarrollador senior especializado en SEO técnico internacional y arquitectura web limpia.
